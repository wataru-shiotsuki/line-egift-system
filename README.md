# LINE eギフトシステム

大手アパレル企業との企業連携PBLとして開発した、LINEプラットフォームを活用したeギフト基盤です。

## 課題
既存のECサイトでは配送が必須となり、若年層との接点が弱く、購入後のデータが顧客理解につながりにくいという課題がありました。

## アプローチ
新規アプリ開発とLINE上で完結させる方法を比較した上で、導入障壁の低さと店舗での受け渡しやすさを踏まえ、LINE上で贈り手が候補を選び、受け取り手が選択できる体験を設計しました。AIとの会話を通じて、誰に何を贈るかを整理する過程から、贈り手の迷いやニーズを取得できる仕組みにしています。

## 主な機能
- AIとの会話による贈答先・贈答シーンのヒアリングとギフト推薦
- 複数候補から受け取り手が選べる選択フロー
- 決済（Stripe）
- 店舗通知・QR受取
- 会話ログ・選択結果・離脱箇所の蓄積

## 技術スタック
- バックエンド：Python, FastAPI
- フロントエンド：React (TypeScript), LIFF
- AI：Gemini API（プロンプトエンジニアリング）
- データベース：Supabase (PostgreSQL)
- 決済：Stripe
- インフラ：Vercel
- 通知：Resend
